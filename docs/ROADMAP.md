# ShopNova 后续扩展路线图

本文档只记录当前未实现或需要生产化增强的功能，不代表当前已完成功能。

## 真实支付接入

- 接入 Stripe、PayPal、支付宝、微信支付或其他真实支付网关。
- 增加支付回调、签名验证、失败重试和支付状态同步。
- 移除完整卡号、CVV 等敏感字段存储。
- 增加退款、部分退款、对账和异常支付处理。

## 图片上传存储

- 接入 Supabase Storage、S3、Cloudflare R2 或其他对象存储。
- 支持后台上传商品图片，而不是手动填写 URL。
- 增加图片压缩、裁剪、格式校验和 CDN 加速。
- 增加图片删除和权限控制。

## 多语言

- 继续完善中、英、日文案覆盖。
- 增加语言选择持久化。
- 支持商品名称、描述、分类和标签的多语言字段。
- 增加日期、货币和地区格式本地化。

## 商品评价

- 新增评价表、评分、评价内容、图片评价和追评。
- 支持用户购买后评价。
- 后台支持评价审核、隐藏和回复。
- 前台商品详情页展示真实评价列表和评分统计。

## 优惠券增强

- 新增 `coupons` 数据表和优惠券后台管理。
- 支持固定金额、百分比折扣、满减、免邮。
- 支持有效期、使用次数、用户限制、商品/分类限制。
- 订单保存优惠券使用记录，避免重复使用。

## 物流跟踪

- 新增发货单、物流公司、运单号和物流轨迹。
- 后台支持录入发货信息。
- 用户端订单详情展示物流进度。
- 可接入第三方物流查询 API。

## 邮件通知

- 注册、下单、支付成功、发货、退款等节点发送邮件。
- 支持邮件模板管理。
- 增加失败重试和发送日志。
- 可扩展短信、站内信和 Web Push。

## 权限系统增强

- 替换当前自建明文认证，接入 Supabase Auth 或后端认证服务。
- 增加密码哈希、邮箱验证、找回密码和多因素认证。
- 后台权限从前端判断升级为服务端/RLS 强校验。
- 细分角色：超级管理员、运营、客服、仓储、财务。

## 数据导出

- 后台订单、支付流水、客户列表支持 CSV 或 Excel 导出。
- 支持按筛选条件导出。
- 增加导出任务记录和下载权限。
- 大数据量时使用异步导出。

## 订单退款流程

- 新增退款申请、退款审核、退款原因和退款状态。
- 支持全额退款和部分退款。
- 和真实支付网关退款接口联动。
- 后台订单详情展示退款历史。

## SEO 优化

- 将后台商品 SEO 字段真正落地到页面 meta 信息。
- 增加商品详情页、分类页的动态标题和描述。
- 增加 sitemap、robots.txt 和结构化数据。
- 支持服务端渲染或预渲染，以增强搜索引擎收录。

- 区分开发、测试、生产环境变量。
- 增加 Supabase 迁移执行流程和回滚说明。
